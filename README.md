# FreeFlow

FreeFlow is a planned, local-first, open-source desktop dictation application for Windows and macOS.

Canonical public repository: [USS-Parks/FreeFlow](https://github.com/USS-Parks/FreeFlow)

The product goal is workflow-level equivalence with modern AI dictation tools: hold a shortcut, speak naturally, and insert clean text into the active application, with local transcription, personalization, recovery, and optional on-device text transformation.

Sequential execution (STS) was authorized on 2026-07-17. Work follows the canonical PSPR in prompt order and does not cross its approval or clean-room boundaries.
