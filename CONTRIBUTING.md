# Contributing to FreeFlow

FreeFlow welcomes focused fixes and PSPR-aligned implementation. Before making
a change, read `AGENTS.md`, the clean-room policy, and the relevant prompt in
`PLANNING/FREEFLOW-CLEAN-ROOM-PSPR.md`.

Contributions must preserve provenance and may not include proprietary Wispr
Flow material or unapproved model weights. Retained or extracted open-source
code needs file-level license and commit attribution.

Run the documented lint, format, frontend, Rust test, provenance, and developer
build gates. Use the issue and pull-request templates and describe any AI
assistance honestly.
